Yam
===
Yet another small module for Drupal that tries to improve the user experience
for content managers, developers and site builders.

Currently provided features:
* In Languages admin page, show the domain associated to each language
* In Features admin page, automatically select a default Feature package
* In Modules admin page, show the path of each module
* In AJAX file uploads, automatically press the 'Upload' form button
* Show extra information in the Status Report
* Creates a menu for module setting pages, and make admin pages available in
  admin/modules/settings/module_name
